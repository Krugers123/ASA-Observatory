# Public Repo Scope

This repository is the public technical edition of ASA Observatory.

Its purpose is to show:
- a working observability engine
- a usable API
- a research dashboard
- example sessions
- trajectory-level and multi-session analysis

This repository is not intended to contain every research artifact, experimental branch, or internal analytical variation connected to the broader ASA / Symbioza program.

Publicly emphasized here:
- drift observation
- coherence tracking
- threshold pressure
- complementarity
- semantic envelope analysis
- dashboard-based forensic inspection

Outside the scope of this public repository may remain:
- unpublished experiments
- private calibration workflows
- additional research modules still under active evaluation
- broader conceptual archives already covered in the Manifest repository

This keeps the repository focused, understandable, and useful as a working technical entry point.
