# SCE

`SCE = Scenario / Counterfactual Engine`

## Purpose

SCE estimates whether stability can still be recovered and how urgent intervention has become.

## What It Measures

In the public surface, SCE is reflected through:

- counterfactual robustness
- anchor lock strength
- intervention priority
- counterfactual state

## How ASA Uses It

ASA uses SCE to move from pure observation toward recovery sensitivity:

- can this trajectory still be re-anchored?
- is the current frame still recoverable?
- how urgent is operator intervention?

## Operator Meaning

SCE helps answer:

- is recovery still realistic?
- how strong is the anchor?
- should the operator act now or continue observing?

## Public Scope

The public edition exposes only the compact operator-facing counterfactual layer.
